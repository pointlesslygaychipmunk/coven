Witch's Coven - Default Page

You're seeing this page because your web server is serving this default.html file instead of correctly routing to the single-page application.

The Witch's Coven application is a React single-page app that needs to route all requests to the main index.html file.

Note for website administrators: You need to configure your server to route all requests to index.html for this single-page application to work properly.

# Apache (.htaccess):
RewriteEngine On
RewriteBase /
RewriteRule ^index\.html$ - [L]
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule . /index.html [L]

# Nginx:
location / {
  try_files $uri $uri/ /index.html;
}
